# 🛒 E-Comm Website

A responsive **eCommerce front-end shopping website** built using **HTML, CSS, and JavaScript**.
This project focuses on clean UI design, responsive layouts, and interactive shopping components to provide a smooth and user-friendly experience.

---

## 📌 Project Overview

This project is a **front-end eCommerce template** that includes common online shopping features such as product listings, special offers, vendor sections, and a cart page.

The goal of this project is to practice **front-end development**, responsive design, and creating an engaging user interface for online stores.

---

## ✨ Features

* Responsive homepage with banner
* Product category listings
* Interactive product cards
* Special offers section
* Vendor showcase section
* Shop page for browsing products
* Cart page for selected items
* Clean and modern UI design
* Mobile-friendly responsive layout

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
